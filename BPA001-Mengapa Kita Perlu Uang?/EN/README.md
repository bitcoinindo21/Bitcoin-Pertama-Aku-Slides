In this first chapter, slides are used to introduce the basic idea of ​​why money is necessary in human life. This section takes readers through the problems that arose when humans exchanged goods directly, leading to the eventual need for a more practical and universal tool. This is where money emerged as a solution to streamline transactions, store value, and consistently measure prices.
